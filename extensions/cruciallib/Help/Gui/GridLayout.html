GridLayoutCalc grid layout calculator


GridLayoutCalc is useful for laying out the larger sections of an interface.


Grid systems are a commonly used method that graphic designers use to layout the structural sections on a page.  You can see this in most any magazine, newspaper or website.  They give a vertical and horizontal rhythm to the design and simplify decisions about where to place things.  It quantizes the visual space.


A GridLayoutCalc is not a decorator like FlowLayout.  You create one and use it directly to calculate bounds, then Composite views or flow views can be placed at those Rects and you use those Rects when placing your views.




(

// a fairly large bounds

b = Window.screenBounds.resizeBy(-100,-100);


w = Window("GridLayoutCalc",b).front;


// create a large grid layout with 12 cols and 7 rows

g = GridLayoutCalc.new(b,12,7);


// specify using row@column Points:  left and bottom right inclusive

// the GridLayoutCalc calculates the rect

TextView(w,g.calc(0@0,6@1)).background = Color.white;


TextView(w,g.calc(0@2,1@9)).background = Color.red;


TextView(w,g.calc(0@10,6@11)).background = Color.black;


TextView(w,g.calc(2@2,6@9)).background = Color.blue;


)


A built in design tool to plan grid layouts

(

// a fairly large bounds

b = Window.screenBounds.resizeBy(-100,-100);


// create a large grid layout with 12 cols and 7 rows

g = GridLayoutCalc.new(b,12,7);

// this shows that the larger dimension (12) consumes as much space as it can

// and that determines the size of each square


// the GUI for the grid object is a tool for planning a layout

g.gui

)





Several ways to specify and obtain bounds.


To get a 2 x 3 box (occupying 6 total grid spaces) in the upper right corner I find it easiest to specify two sets of quadrants that encompass the area.


// top left and bottom right corner

g.calc( 0@0, 2@1 )

// you can specify any two quadrants in any order


// get the same result but specifying by quadrant arrays

g.getBounds( [0,0],[2,1] )


// or specify using a rect with the left,top, width, height in integers

g.mapRect( Rect(0,0,2,3) )



(

b = GUI.window.screenBounds.resizeBy(-100,-100);

g = GridLayoutCalc.new(b,12,7);

w = GUI.window.new(b).front;


// 12 cols and as many rows as will fit within the bounds

g = GridLayoutCalc.new(b,12);


)
